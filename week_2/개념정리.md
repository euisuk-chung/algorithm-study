## 자료구조 기초 개념

### Array(배열) and Linked List(리스트)

|Case|Array|Linked List|
|:------:|:------:|:------:|
|특정 원소 조회|O(1)|O(N)|
|중간 삽입 및 삭제|O(N)|O(1)|
|데이터 추가|데이터가 추가 시 모든 공간이 다 차면 새로운 공간을 할당 받아야함|모든 공간이 다 찼어도 맨 뒤의 노드만 동적으로 추가하면 됨|
|정리|덷이터에 접근하는 경우가 빈번 시 Array 사용|삽입과 삭제가 빈번하다면 Listed List를 사용하는 것이 좋음|

- `참고` : Python 의 list 는 array 로 구현되어 있지만, Python 의 list 는 array 로 구현되어 있습니다. 이를 통해 파이썬의 배열은 링크드 리스트로 쓸 수도 있고, 
배열로도 쓸 수 있게 만든 효율적인 자료구조다! 라고만 생각해주시면 됩니다.

### Class(클래스)
**Class(클래스)란 무엇인가?**
- 클래스는 분류. 집합. 같은 속성과 기능을 가진 객체를 총칭하는 개념입니다.

**Object(객체)란 무엇인가?**
- 객체는 세상에 존재하는 유일무이한 사물입니다.

**Instance(인스턴스)란 무엇인가?**
- 클래스로 만든 객체를 인스턴스라고도 한다. 

**예시**
- 클래스가 사람이라면, 객체는 유재석이 될수도 있고, 박명수가 될 수도 있습니다.
- 클래스가 동물이라면, 객체는 강아지가 될수도 있고, 고양이가 될 수도 있습니다.
- 유재석 객체의 사람의 인스턴스고, 강아지는 동물의 인스턴스입니다.

**Constructor(생성자)란 무엇인가?**
- 클래스에는 생성자(Constructor)는 객체를 생성할 때 데이터를 넣어주거나, 내부적으로 원하는 행동을 실행하게 할 수 있습니다!
- 파이썬에서 생성자 함수의 이름은 `__init__` 으로 고정되어 있고, 생성시에 호출되는 함수입니다.

**Method(메소드)란 무엇인가?**
- 클래스 내부의 함수는 메소드(method) 라고 부릅니다.

**Inheritance(상속)이란 무엇인가?**
- 상속(Inheritance)이란 "물려받다"라는 뜻으로, "재산을 상속받다"라고 할 때의 상속과 같은 의미입니다
- 클래스에도 이 개념을 적용할 수 있다. 어떤 클래스를 만들 때 다른 클래스의 기능을 물려받을 수 있게 만드는 것입니다. 
- 보통 상속은 기존 클래스를 변경하지 않고 기능을 추가하거나 기존 기능을 변경하려고 할 때 사용합니다.
